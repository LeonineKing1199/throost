makeZipIterator(…args) → {ZipIterator}

Iterate over a variadic number of iterators in a single pass. Useful for creating a structure of arrays vs an array of structures approach

Each dereference of the iterators yields an array of the values, where the order of the values matches the order of the supplied iterators as they were passed in
Parameters:
Name Type Attributes Description
args Iterators <repeatable>
 Iterators to simultaneously traverse
Source:
  • make-zip-iterator.js, line 47
Returns:
Iterator over several sequences
Type
ZipIterator
Example
const x = [0, 1, 2, 3];
const y = new Set([4, 5, 6, 7]);
const z = function*() {
  for (let i = 0; i < 4; ++i) yield i * i; 
};

const it = makeZipIterator(begin(x), begin(y), z());

// prints
// [0, 4, 0]
// [1, 5, 1]
