genToIterable(genFn) → {Iterable}

Unfortunately, almost everything in JavaScript has a Symbol.iterator property. However, generators do not!

This is a helper function that creates an iterable-conforming interface out of a generator function. The iterable type will forward all supplied arguments to the generator function

makeZipIterator() → {ZipIterator}

Iterate over a variadic number of iterators in a single pass. Useful for creating a structure of arrays vs an array of structures
